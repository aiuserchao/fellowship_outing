# 团契活动 - Cuyahoga Valley National Park 2026年7月独立日出行计划

本仓库包含小羊团契组织的2026年7月独立日(Cuyahoga Valley National Park)出行活动的详细行程安排。

## 当前行程版本

- **[2日行程示例_团契活动_CuyahogaValley.md](./2日行程示例_团契活动_CuyahogaValley.md)** - 详细的两日行程计划（7月3日-4日）
  - 已确认酒店：Staybridge Suites Akron-Stow-Cuyahoga Falls by IHG 和 Home2 Suites by Hilton Stow Akron
  - 特别考虑：包含3-12岁儿童及15个月婴孩的家庭活动选项
  - 灵活安排：家庭可根据小孩年龄和精力自行调整参与程度

## 活动概述

- **目的地**：Cuyahoga Valley National Park，俄亥俄州
- **时间**：2026年7月3日周五下午 - 7月4日周日下午
- **交通**：自驾（建议拼车以增进团契）
- **住宿**：园区附近酒店（需提前预约，注意婴儿床/儿童设施）
- **特别注意**：7月4日为美国独立日，园区将非常繁忙，需早到并预约野餐遮蔽所

## 行程亮点

1. **已确认住宿**：提供两家家庭友好型酒店选择
2. **婴幼儿友好**：特别安排适合15个月婴儿及3-12岁儿童的活动
3. **独立日体验**：7月4日在园区庆祝美国独立日
4. **多样化活动**：从轻松散步到中等难度徒步，包含景观铁路和瀑布观赏
5. **团契时间**：专门安排分享、祷告和敬拜时间
6. **安全考虑**：详细的必备物品清单、应急准备和儿童看护建议

## 如何使用

1. 查看[详细行程](./2日行程示例_团契活动_CuyahogaValley.md)了解完整安排
2. 根据您家庭的具体情况调整参与程度
3. 提前预订所选酒店（尤其是7月4日周末）
4. 准备必备物品清单中列出的物品
